#### _Naming conventions_

- Assets (images, icons, etc) --> `kebab-case` --> eg. `smartapes-logo.png`
- React components --> `PascalCase` --> eg. `Dashboard.js`
- Javascript helper files --> `camelCase` --> eg. `api.js`
- Custom hooks --> `use + PascalCase` --> eg. `useNotification.js`

#### _Coding conventions_

Use best practices, for example take a look at AirBnb Javascript coding style: `https://github.com/airbnb/javascript`
